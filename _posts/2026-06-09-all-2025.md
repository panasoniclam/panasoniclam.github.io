# Danh sách tính năng đã phát triển

 

## 1. Annual Goals — Mục tiêu năm

### 1.1 Goal Management (Quản lý mục tiêu)
- **Danh sách mục tiêu** — `/annual-goals/goals`
- **Đăng ký / chi tiết mục tiêu** — `/annual-goals/register`, `/annual-goals/detail`
- **Mục tiêu nhân viên** — `/annual-goals/empgoal`
- **Phê duyệt mục tiêu** — `/annual-goals/approval`
- **Measurement (đo lường)** — `/annual-goals/measurement`, `what-measurement`, `how-measurement`
- **Calibration mục tiêu** — `/annual-goals/calibration`

### 1.2 PMS Check-in (Đánh giá kết quả)
- **Check-in list & detail** — `/annual-goals/checkin`, `/annual-goals/checkin/detail`
- **Check-in WHAT** (kết quả công việc) — `/annual-goals/checkin/what`
  - Hỗ trợ đo theo `metric / progress / yes-no`
  - Tự tính tiến độ theo direction: At most / At least / Downward / In range
  - Cap tiến độ ở 100% cho hướng "Tối đa" (cả khi direction là chuỗi)
- **Check-in HOW** (cách thực hiện) — `/annual-goals/checkin/how`
- **Phê duyệt check-in** — `/annual-goals/checkin/approve`
- Manager copy data từ employee, lưu `checkinDetailAchievement = checkinDetailProgress` (chỉ WHAT)

### 1.3 IDP — Individual Development Plan
- **Danh sách IDP** — `/annual-goals/empidp`, `/annual-goals/idp-detail`
- **IDP Check-in flow** — `/annual-goals/idp-checkin`
  - `idp-checkin/detail` — chi tiết check-in IDP
  - `idp-checkin/item` — check-in từng item (Experience / Exposure / Education / Next Action / Timeline / Support)
  - `idp-checkin/approve-list` — duyệt IDP check-in
- Validate file đính kèm `.xlsx`, upload có timestamp suffix
- Dùng theme `has-data-textarea` cho input có dữ liệu (đồng bộ UI)

## 19. Đa ngôn ngữ (i18n)

- Hỗ trợ **Vietnamese (vi) / English (en) / Chinese (cn)**
- Backend trả các field `appNameCn` / `featureNameCn` / `fieldNameCn` (backward-compatible)
- Helper `getLocalizedLabel(language, name, nameEn, nameCn)` với fallback CN → EN → VI
- Áp dụng cho: menu chính, role management, query builder, dropdown filter
- Selector ngôn ngữ có option 中文

## Ghi chú

- Các tính năng được phát triển song song nhiều module, mỗi module thường có pattern: **list / detail / register / approve / behalf / import / config**.
- Toàn bộ tính năng sử dụng theme Nebular (`nb-card`, `nb-button`, `nb-input`, …) và i18n service `appLanguageService`.
- Các fix bug và cải tiến UI (UI checkin, có data highlight, validate file, cap tiến độ, format số, …) đã được tích hợp xuyên suốt các module.
